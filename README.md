The UN General Debate Corpus can be downloaded here: https://www.ungdc.bham.ac.uk/

A text as data approach to the UNGDC, based on the class "Text as Data" (POLI 176) at UCSD.
The code used to actually process the corpus is in Bartt_FinalCode.r.
The handcoding was used to train the model and use a supervised ML approach to judging whether a text was critical of the structure of the UN itself. Please contact me for the full report if interested.
